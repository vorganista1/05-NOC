# Proyecto NOC


1. clonar el archivo env.template a .env
2. Condifurar las variables de entorno 
```````````````
PORT = 3000

MAILER_EMAIL = 
MAILER_SECRET_KEY = 

PROD = false
```````````````

3. Ejecutar el comando `
`` npm install``

4. leantar las bases de datos con el comando 
 `````
 docker compose up -d
 ````
4. Ejecutar ``npm run dev``
